# BR-002 Text box does not scroll when content exceeds box

Jira bug template

- Summary: Text box does not scroll when content exceeds box
- Environment: Brave 112.0 on macOS
- Severity: Medium
- Priority: Normal
- Type: UI
- Status: New
- Reporter: QA
- Reproducibility: Always
- Test Data: N/A

## Preconditions
None

## Steps to Reproduce
1. Go to the text box page
2. Add content until it exceeds the box size
3. Attempt to scroll inside the text box

## Expected Result
Text box scrolls while page remains stable.

## Actual Result
Whole page scrolls; text box does not.
